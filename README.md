# Mali-GPU-Performance-Capture

## What is this?

This is a repo documenting what I do for analysing GPU performance on Mali GPU based android devices.

## Who am I?

I am an ex-ARM engineer who had been working as part of the Mali GPU software team for more than 5 years.

## What you can learn from this repo?

I will write down a workflow which tells you how to analyse GPU performance based on Android and Vulkan applications.



## Prerequisites

You need an environment for building Android applications.
